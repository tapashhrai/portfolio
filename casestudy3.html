The Product

This cross platform product allow users to explore and know more about holy places in Bhutan. These places are divide according to zone wise which will be easier for user to know and search holy places.

PROBLEM

Most of the people does not know about holy place that exist within Bhutan.

THE GOAL

The goal of this project is to create a cross platform where user will be able to explore and know more about the holy places in Bhutan.

ROLE

Responsibilities

UNDERSTANDING THE USER

User research: Summary

Conducted user interviews to better understand the target and user needs. After conducting the user interviews, the primary users group was working adults and parents who loves to visit holy places. This platform can be also used by a tourist who like to travel and visit spiritual places. Before they visit holy places they want to know little bit about the holy place like which district it belongs.

DESIGN

Paper wireframe

I sketched paper wireframe of homepage where users can view holyplace with location on top. The also added little description about the app in about us page, then incorporated those ideas into the digital low fidelity wireframes.



Digital wireframe

Next, I moved to Adobe XD to create digital wireframes for all of the screens. Using a basic design system, I was able to create low fidelity wireframes that were still clean and organized. The homepage serves as a central hub where users can access a quick dashboard. The navigation gives users options to view holy pace and about us.
